# Bhromaon
Create user interactive travel website 
# Project Overview

Bhromaon is a modern, responsive travel website built with HTML, SCSS, and JavaScript. The site allows users to explore destinations, view services, and enjoy interactive animations and smooth scrolling.

The project demonstrates front-end best practices including responsive design, flex/grid layouts, and dynamic interactivity using JavaScript.

# Technologies Used

* HTML5 – Structure of the website.

* CSS3 / SCSS (SASS) – Styling, variables, nesting, and modular CSS.

* JavaScript – Smooth scrolling, navigation menu, and interactive elements.

* Font Awesome – Icons for social media and buttons.

* Animate.css – Prebuilt CSS animations.

* Google Fonts – Josefin Sans & Aleo for typography.

* Normalize.css – Browser style consistency.
